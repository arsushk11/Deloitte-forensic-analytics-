# 🔍 Deloitte Data Analytics Job Simulation

## 📌 Project Overview
Completed Deloitte's Data Analytics Job Simulation on Forage (April 2026).

This simulation involved two real-world forensic analytics tasks:
1. **Telemetry Analysis** — Analyzing IoT device data from Daikibo's global factories
2. **Equality Analysis** — Classifying gender pay equity across job roles using Excel

---

## 🛠️ Tools Used
- Tableau (Telemetry Dashboards)
- Microsoft Excel (Equality Classification Model)
- JSON (Semi-structured data source)

---
