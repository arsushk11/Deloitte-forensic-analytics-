# 🔍 Deloitte Data Analytics Job Simulation

## 📌 Project Overview
Completed Deloitte's Data Analytics Job Simulation on Forage (April 2026).

This simulation involved two real-world forensic analytics tasks:
1. **Telemetry Analysis** — Analyzing IoT device data from Daikibo's global factories
2. **Equality Analysis** — Classifying gender pay equity across job roles using Excel

---

## 🛠️ Tools Used
- Tableau (Telemetry Dashboards)
- Microsoft Excel (Equality Classification Model)
- JSON (Semi-structured data source)

---

## 📊 Task 1 — Telemetry Data Analysis

**Goal:** Identify which machines are breaking down the most and in which factories.

**Data Fields:** deviceID, deviceType, temperature, status, factory, city, country, timestamp

**Dashboards Built:**
- Down Time per Device Type
- Down Time per Factory

**Key Findings:**
- Analyzed device failure patterns across 4 global factories
- Identified highest-downtime device types to support predictive maintenance
- Connected semi-structured JSON data directly into Tableau

---

## 📊 Task 2 — Workforce Equality Analysis

**Goal:** Classify gender pay equality scores for each job role across all factories.

**Formula Used:**
=IF(ABS(C2)<=10, "Fair", IF(ABS(C2)<=20, "Unfair", "Highly Discriminative"))
**Factories Analyzed:**
- Daikibo Factory Meiyo
- Daikibo Factory Seiko
- Daikibo Berlin
- Daikibo Shenzhen

**Job Roles Covered:** C-Level, VP, Director, Sr. Manager, Manager, Jr. Manager, Sr. Engineer, Engineer, Jr. Engineer, Operational Support, Machine Operator

**Key Finding:** C-Level and VP positions showed "Highly Discriminative" pay gaps across multiple factories.
